Схема дерева в примере:
            8
         /     \
        3       10
      /   \       \
     1     6      14
      \   / \    /  \
       2 4   7  13   15
          \           \
           5           18
8 - это корень дерева;
3,10,6,14 - узлы дерева;
1,4,7,13 - листья дерева;
между узлами есть связи: потомки-родители, братья (sibling);
для узла 3 родителем является 8, а для 6 родитель 3;
у узла 3 есть потомки 1 и 6;
у 1 есть брат (sibling) 6, у 4 есть есть брат 7
